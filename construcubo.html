Project

CONSTRUCUBO

TECH STACK: HTML, CSS, JAVASCRIPT, REACT, REDUX.

Responsive website for ConstruCubo, a construction and real estate development company based in Medellin, Colombia. The website features animations on scroll and hover, custom fonts, navigation menu, features menu, contact form, and a search feature that enables users to find properties for sale and save them under favorites.

Aditional features include: sign up for a new account, login to an existing account, mark properties as favorites, filter by location and/or price, and comment under the properties' listings.
