Project

MONARCHA CONSULTING

TECH STACK: HTML, CSS, JAVASCRIPT, PYTHON, DJANGO.

I built a website with a fresh and modern look for Monarcha Consulting, a ficticious digital marketing agency. It took me two months to complete while working full time.

The website features animations on scroll and hover, custom fonts, navigation menu, features menu, FAQ section with expandable items, contact form, and a basic yet robust admin console that requires login and enables users to add, delete, and edit leads, articles, and books.

Aditional features of the admin console include the ability to select custom fields such as Lead Stage, Services, Packages, and General Information. The site uses Django models to store, retrieve, update and/or delete data. Every single line of code, function and feature was built from scratch using the technologies mentioned above.
